22  or a list of images all of the same size.
23  nrow (int, optional): Number of images displayed in each row of the grid.
24  The final grid size is ``(B / nrow, nrow)``. Default: ``8``.
25  padding (int, optional): amount of padding. Default: ``2``.
26  normalize (bool, optional): If True, shift the image to the range (0, 1),
27  by the min and max values specified by :attr:`range`. Default: ``False``.
28  range (tuple, optional): tuple (min, max) where min and max are numbers,
29  then these numbers are used to normalize the image. By default, min and max
30  are computed from the tensor.
31  scale_each (bool, optional): If ``True``, scale each image in the batch of
32  images separately rather than the (min, max) over all images. Default: ``False``.
33  pad_value (float, optional): Value for the padded pixels. Default: ``0``.
34 
35  Example:
36  See this notebook `here <https://gist.github.com/anonymous/bf16430f7750c023141c562f3e9f2a91>`_
