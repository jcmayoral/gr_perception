11  The data-format is : [label [index:value ]*256 \\n] * num_lines, where ``label`` lies in ``[1, 10]``.
12  The value for each pixel lies in ``[-1, 1]``. Here we transform the ``label`` into ``[0, 9]``
13  and make pixel values in ``[0, 255]``.
14 
15  Args:

21  target_transform (callable, optional): A function/transform that takes in the

23  download (bool, optional): If true, downloads the dataset from the internet and

25  downloaded again.
26 
27  """
