20 class MaskRCNN(FasterRCNN):
21  """
22  Implements Mask R-CNN.
23 
24  The input to the model is expected to be a list of tensors, each of shape [C, H, W], one for each
25  image, and should be in 0-1 range. Different images can have different sizes.
26 
27  The behavior of the model changes depending if it is in training or evaluation mode.
28 
29  During training, the model expects both the input tensors, as well as a targets (list of dictionary),
30  containing:
31  - boxes (FloatTensor[N, 4]): the ground-truth boxes in [x1, y1, x2, y2] format, with values of x
32  between 0 and W and values of y between 0 and H
33  - labels (Int64Tensor[N]): the class label for each ground-truth box
34  - masks (UInt8Tensor[N, H, W]): the segmentation binary masks for each instance
35 
36  The model returns a Dict[Tensor] during training, containing the classification and regression
37  losses for both the RPN and the R-CNN, and the mask loss.
38 
39  During inference, the model requires only the input tensors, and returns the post-processed
40  predictions as a List[Dict[Tensor]], one for each input image. The fields of the Dict are as
41  follows:
42  - boxes (FloatTensor[N, 4]): the predicted boxes in [x1, y1, x2, y2] format, with values of x
43  between 0 and W and values of y between 0 and H
44  - labels (Int64Tensor[N]): the predicted labels for each image
45  - scores (Tensor[N]): the scores or each prediction
46  - masks (UInt8Tensor[N, 1, H, W]): the predicted masks for each instance, in 0-1 range. In order to
47  obtain the final segmentation masks, the soft masks can be thresholded, generally
48  with a value of 0.5 (mask >= 0.5)
49 
50  Arguments:
51  backbone (nn.Module): the network used to compute the features for the model.
52  It should contain a out_channels attribute, which indicates the number of output
53  channels that each feature map has (and it should be the same for all feature maps).
54  The backbone should return a single Tensor or and OrderedDict[Tensor].
55  num_classes (int): number of output classes of the model (including the background).
56  If box_predictor is specified, num_classes should be None.
57  min_size (int): minimum size of the image to be rescaled before feeding it to the backbone
58  max_size (int): maximum size of the image to be rescaled before feeding it to the backbone
59  image_mean (Tuple[float, float, float]): mean values used for input normalization.
60  They are generally the mean values of the dataset on which the backbone has been trained
61  on
62  image_std (Tuple[float, float, float]): std values used for input normalization.
63  They are generally the std values of the dataset on which the backbone has been trained on
64  rpn_anchor_generator (AnchorGenerator): module that generates the anchors for a set of feature
65  maps.
66  rpn_head (nn.Module): module that computes the objectness and regression deltas from the RPN
67  rpn_pre_nms_top_n_train (int): number of proposals to keep before applying NMS during training
68  rpn_pre_nms_top_n_test (int): number of proposals to keep before applying NMS during testing
69  rpn_post_nms_top_n_train (int): number of proposals to keep after applying NMS during training
70  rpn_post_nms_top_n_test (int): number of proposals to keep after applying NMS during testing
71  rpn_nms_thresh (float): NMS threshold used for postprocessing the RPN proposals
72  rpn_fg_iou_thresh (float): minimum IoU between the anchor and the GT box so that they can be
73  considered as positive during training of the RPN.
74  rpn_bg_iou_thresh (float): maximum IoU between the anchor and the GT box so that they can be
75  considered as negative during training of the RPN.
76  rpn_batch_size_per_image (int): number of anchors that are sampled during training of the RPN
77  for computing the loss
78  rpn_positive_fraction (float): proportion of positive anchors in a mini-batch during training
79  of the RPN
80  box_roi_pool (MultiScaleRoIAlign): the module which crops and resizes the feature maps in
81  the locations indicated by the bounding boxes
82  box_head (nn.Module): module that takes the cropped feature maps as input
83  box_predictor (nn.Module): module that takes the output of box_head and returns the
84  classification logits and box regression deltas.
85  box_score_thresh (float): during inference, only return proposals with a classification score
86  greater than box_score_thresh
87  box_nms_thresh (float): NMS threshold for the prediction head. Used during inference
88  box_detections_per_img (int): maximum number of detections per image, for all classes.
89  box_fg_iou_thresh (float): minimum IoU between the proposals and the GT box so that they can be
90  considered as positive during training of the classification head
91  box_bg_iou_thresh (float): maximum IoU between the proposals and the GT box so that they can be
92  considered as negative during training of the classification head
93  box_batch_size_per_image (int): number of proposals that are sampled during training of the
94  classification head
95  box_positive_fraction (float): proportion of positive proposals in a mini-batch during training
96  of the classification head
97  bbox_reg_weights (Tuple[float, float, float, float]): weights for the encoding/decoding of the
98  bounding boxes
99  mask_roi_pool (MultiScaleRoIAlign): the module which crops and resizes the feature maps in
100  the locations indicated by the bounding boxes, which will be used for the mask head.
101  mask_head (nn.Module): module that takes the cropped feature maps as input
102  mask_predictor (nn.Module): module that takes the output of the mask_head and returns the
103  segmentation mask logits

124  >>> # aspect ratios

126  >>> aspect_ratios=((0.5, 1.0, 2.0),))
127  >>>
128  >>> # let's define what are the feature maps that we will
129  >>> # use to perform the region of interest cropping, as well as
130  >>> # the size of the crop after rescaling.
131  >>> # if your backbone returns a Tensor, featmap_names is expected to
132  >>> # be ['0']. More generally, the backbone should return an
133  >>> # OrderedDict[Tensor], and in featmap_names you can choose which
134  >>> # feature maps to use.

136  >>> output_size=7,
137  >>> sampling_ratio=2)
138  >>>
139  >>> mask_roi_pooler = torchvision.ops.MultiScaleRoIAlign(featmap_names=['0'],
140  >>> output_size=14,
141  >>> sampling_ratio=2)
142  >>> # put the pieces together inside a MaskRCNN model
143  >>> model = MaskRCNN(backbone,
144  >>> num_classes=2,

146  >>> box_roi_pool=roi_pooler,
147  >>> mask_roi_pool=mask_roi_pooler)

149  >>> x = [torch.rand(3, 300, 400), torch.rand(3, 500, 400)]

221 class MaskRCNNHeads(nn.Sequential):
222  def __init__(self, in_channels, layers, dilation):
223  """
224  Arguments:
225  in_channels (int): number of input channels
226  layers (list): feature dimensions of each FCN layer
227  dilation (int): dilation rate of kernel
228  """
229  d = OrderedDict()
230  next_feature = in_channels
231  for layer_idx, layer_features in enumerate(layers, 1):
232  d["mask_fcn{}".format(layer_idx)] = nn.Conv2d(
233  next_feature, layer_features, kernel_size=3,
234  stride=1, padding=dilation, dilation=dilation)
235  d["relu{}".format(layer_idx)] = nn.ReLU(inplace=True)
236  next_feature = layer_features
237 
238  super(MaskRCNNHeads, self).__init__(d)
239  for name, param in self.named_parameters():
240  if "weight" in name:
241  nn.init.kaiming_normal_(param, mode="fan_out", nonlinearity="relu")
242  # elif "bias" in name:

267 def maskrcnn_resnet50_fpn(pretrained=False, progress=True,
268  num_classes=91, pretrained_backbone=True, trainable_backbone_layers=3, **kwargs):
269  """
270  Constructs a Mask R-CNN model with a ResNet-50-FPN backbone.
271 
272  The input to the model is expected to be a list of tensors, each of shape ``[C, H, W]``, one for each
273  image, and should be in ``0-1`` range. Different images can have different sizes.
274 
275  The behavior of the model changes depending if it is in training or evaluation mode.
276 
277  During training, the model expects both the input tensors, as well as a targets (list of dictionary),
278  containing:
279  - boxes (``FloatTensor[N, 4]``): the ground-truth boxes in ``[x1, y1, x2, y2]`` format, with values of ``x``
280  between ``0`` and ``W`` and values of ``y`` between ``0`` and ``H``
281  - labels (``Int64Tensor[N]``): the class label for each ground-truth box
282  - masks (``UInt8Tensor[N, H, W]``): the segmentation binary masks for each instance
283 
284  The model returns a ``Dict[Tensor]`` during training, containing the classification and regression
285  losses for both the RPN and the R-CNN, and the mask loss.
286 
287  During inference, the model requires only the input tensors, and returns the post-processed
288  predictions as a ``List[Dict[Tensor]]``, one for each input image. The fields of the ``Dict`` are as
289  follows:
290  - boxes (``FloatTensor[N, 4]``): the predicted boxes in ``[x1, y1, x2, y2]`` format, with values of ``x``
291  between ``0`` and ``W`` and values of ``y`` between ``0`` and ``H``
292  - labels (``Int64Tensor[N]``): the predicted labels for each image
293  - scores (``Tensor[N]``): the scores or each prediction
294  - masks (``UInt8Tensor[N, 1, H, W]``): the predicted masks for each instance, in ``0-1`` range. In order to
295  obtain the final segmentation masks, the soft masks can be thresholded, generally
296  with a value of 0.5 (``mask >= 0.5``)
297 
298  Mask R-CNN is exportable to ONNX for a fixed batch size with inputs images of fixed size.
299 
300  Example::
301 
302  >>> model = torchvision.models.detection.maskrcnn_resnet50_fpn(pretrained=True)
303  >>> model.eval()
304  >>> x = [torch.rand(3, 300, 400), torch.rand(3, 500, 400)]
305  >>> predictions = model(x)
306  >>>
307  >>> # optionally, if you want to export the model to ONNX:
308  >>> torch.onnx.export(model, x, "mask_rcnn.onnx", opset_version = 11)
309 
310  Arguments:
311  pretrained (bool): If True, returns a model pre-trained on COCO train2017
312  progress (bool): If True, displays a progress bar of the download to stderr
313  pretrained_backbone (bool): If True, returns a model with backbone pre-trained on Imagenet
314  num_classes (int): number of output classes of the model (including the background)
315  trainable_backbone_layers (int): number of trainable (not frozen) resnet layers starting from final block.
316  Valid values are between 0 and 5, with 5 meaning all backbone layers are trainable.
317  """
318  assert trainable_backbone_layers <= 5 and trainable_backbone_layers >= 0
319  # dont freeze any layers if pretrained model or backbone is not used
320  if not (pretrained or pretrained_backbone):
321  trainable_backbone_layers = 5
322  if pretrained:
323  # no need to download the backbone if pretrained is set
324  pretrained_backbone = False
325  backbone = resnet_fpn_backbone('resnet50', pretrained_backbone, trainable_layers=trainable_backbone_layers)
326  model = MaskRCNN(backbone, num_classes, **kwargs)
327  if pretrained:
328  state_dict = load_state_dict_from_url(model_urls['maskrcnn_resnet50_fpn_coco'],
329  progress=progress)
330  model.load_state_dict(state_dict)
331  return model
